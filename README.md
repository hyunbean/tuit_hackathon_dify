# tuit-hackathon-dify

BozorCheck AI — 글로벌 해커톤(국민대학교 x TUIT, 우즈베키스탄) 팀 프로젝트에서
Dify 파트를 담당하며 설계·구축한 3개의 에이전트 워크플로우 DSL export.

프론트엔드/백엔드는 팀원이 별도로 개발했으며, 이 저장소에는 담당 파트인
Dify 워크플로우 설정 파일만 포함되어 있습니다.

## 구성

- `BozorCheck_Price_Insight_v1.yml` — 재래시장 시세 인사이트 에이전트
- `BozorCheck_Product_Normalizer_v1.yml` — 상품명 정규화 에이전트
- `BozorCheck_Report_Inspector_v1.yml` — 신고/검수 에이전트

## 결과

글로벌 해커톤 3등 (BozorCheck AI — 재래시장 실시간 시세 조회 서비스)
